# Amigo Secreto

## Descrição
Meu projeto final da turma de Iniciante em Programação G8 - ONE:
Um código em JavaScript que faz um sorteio de Amigo Secreto. 
Uma vez que os nomes dos participantes são adicionados, o sistema faz o sorteio automaticamente, garantindo que nenhuma pessoa tire a si mesma.

## Funcionalidades
- Adicionar nomes de participantes
- Realizar o sorteio de forma aleatória
- Garantir que nenhum participante tire a si mesmo
- Exibir os pares sorteados

## Tecnologias Utilizadas
- HTML
- CSS
- JavaScript

## Como Usar
1. Clone este repositório:
   ```bash
   git clone https://github.com/kguimaraes/challenge-amigo-secreto_pt
   ```
2. Abra o arquivo `index.html` em um navegador.
3. Adicione os participantes no campo de entrada e clique em "Adicionar".
4. Quando todos os participantes forem adicionados, clique em "Sortear".
5. O resultado do sorteio será exibido na tela.
